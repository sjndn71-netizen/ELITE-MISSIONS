### Testing Checklist

- [ ] Tested on private server
- [ ] Tested on alt account
- [ ] No crashes found
- [ ] All 3 missions work
- [ ] No new errors in console
- [ ] Performance is acceptable
- [ ] Settings don't break script
- [ ] Auto-loop doesn't spam

## Commit Message Format

```
Type: Subject line (max 50 chars)

Optional longer description explaining the change.
Why the change was made, not what was changed.

Fix: #123  (reference issue number if applicable)
```

### Types
- `Fix:` Bug fixes
- `Add:` New features
- `Improve:` Code improvements
- `Refactor:` Code reorganization
- `Docs:` Documentation changes
